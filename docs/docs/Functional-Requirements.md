# Functional Requirements

## User Management
- Users must be able to register and create secure accounts.
- Users must be able to log in and log out securely.
- Role-based access control (Patient, Doctor, Admin).

## Appointment & Telehealth
- Patients can book virtual consultations.
- Doctors can accept or reschedule appointments.
- System sends automated appointment reminders.
- Video consultation functionality must be integrated.

## AI Symptom Assessment
- Users can complete a symptom questionnaire.
- System provides preliminary triage guidance.
- AI results are visible to healthcare providers before consultation.

## Patient Records
- Doctors can access patient medical history.
- Patients can upload medical documents.
- Records must be searchable and filterable.

## Offline Capability
- Users can access core features offline.
- Data must sync automatically once connectivity is restored.
